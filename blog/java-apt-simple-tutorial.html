2. 重点说调试部分

需要把javac设置为debug模式,在IntelliJ IDEA 13里面设置步骤为
File—-Settings—-Compiler—-Additional build process VM options
添加-Xdebug -Xrunjdwp:transport=dt_socket,server=y,suspend=y,address=5003

然后建一个远程调试就行了.
