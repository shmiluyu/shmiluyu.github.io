About Spring Cloud

发表于 2017-03-23 | 分类于 spring cloud

本文是根据Udemy上的Spring Cloud教程整理而成.算是做个笔记吧

Spring Cloud 起源

  • 首先是在2004年的时候,出现了第一版的SpringFramework,被称之为EJB的替代者
  • 2006-至今,Spring拆分成了多个子项目,涵盖了Java开发的方方面面.最后统一在SpringIO旗下

Spring Cloud 子项目

  • Spring Cloud Config
  • Spring Cloud Netflix
    • Eureka
    • Hystrix
    • Zuul
    • Ribbon
    • Feign
  • Spring Cloud Bus
  • etc..

Spring Cloud 可以做到

  • 分布式配置管理
  • 服务注册与发现
  • 路由(我理解的是ApiGateway)
  • 点对点调用
  • 负载均衡
  • 服务熔断
  • 全局锁(我理解的是分布式锁)
  • 集群Leader选举和状态监控
  • 分布式消息
  • ect…

关于微服务的 服务端 和 客户端

  • 服务端和客户端是相对而言的
  • 一个微服务经常即是服务端也是客户端
  • 不要被这些词汇搞混了
